Mashov Data Fetcher

Welcome to Mashov Data Fetcher

This tool allows you to easily fetch data from the Mashov website and display it in a user-friendly format.

Simply upload a JSON file containing the input data specified, and the tool will do the rest.

Here Is what the JSON file should look like:
                {
                    "SEMEL": "XXXXXX", 
                    "YEAR": 2024/Current YEAR,
                    "ID": "XXXXXX",
                    "PASSWORD": "XXXXXX"
                }

To download this format click Here

Heres how to get the SEMEL: go to the mashov website and login, you will see school name and the SEMEL in the topleft corner: If you Dont know your PASSWORD/ID ask your teacher/school admin for help.
